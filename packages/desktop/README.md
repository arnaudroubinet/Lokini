# @lokini/desktop

Application desktop via Tauri — **post-MVP**.

Ce package contiendra le wrapper Tauri utilisant l'application web (`@lokini/web`) dans une WebView native. Binaire compact et performant pour macOS, Windows et Linux.

## Stack prévue

- Tauri v2
- @lokini/web (encapsulé dans la WebView)
- @lokini/core (logique métier partagée)
